¿Qué es Javascript y en qué se asemeja a Java?

Javascript es un lenguaje de programación multiplataforma, orientada a objetos basada en la especificación llamado o ECMAScript estandarizado por Ecma International.

Además de HTML y CSS, Javascript es una de las tecnologías fundamamentales de la WWW (World Wide Web). Casi el 97% de las páginas usa su lado-cliente para el comportamiento de la página.

Javascript y Java ambos tienen soporte multiplataforma, en Java tiene JVM (Máquina Virtual Java) disponible para muchos sistemas operativos, y en Javascript porque los navegadores web están disponibles para todos los usuarios de todos los sistemas operativos.
